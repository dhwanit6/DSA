# 🤝 Contributing

Found a broken link? Know a better free resource? Caught an error? PRs are welcome.

## Guidelines

1. **Content must be free.** Do not recommend paid courses or resources.
2. **Problems must earn their place.** Every problem added must cover a pattern not already covered. No padding.
3. **Cross-validate claims.** If you add interview advice, cite a source (interview report, published data, named authority).
4. **Maintain the structure.** Files go in the correct directory. Links must resolve. Follow existing formatting.
5. **No placeholder content.** Every section must be complete and production-quality.

## How to Contribute

1. Fork this repo
2. Create a branch: `git checkout -b feature/your-addition`
3. Make your changes
4. Test all links: ensure every `[link](path)` in your changes resolves to a real file
5. Submit a PR with a description of what you changed and why
