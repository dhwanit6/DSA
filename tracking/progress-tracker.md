# Progress Tracker - Spaced Repetition System

> If you do nothing else, keep this tracker updated. It is the biggest multiplier.

---

## 1) Why It Works

Memory fades fast. Reviews on Day 3, 7, and 21 lock it in.

---

## 2) Setup Columns (Google Sheets or Notion)

- Problem name
- Platform ID
- Topic
- Pattern
- Difficulty
- Date solved
- Review dates (D3, D7, D21)
- Done checkmarks
- Owned (Y/N)
- Key insight

---

## 3) Review Protocol

1. Day 3: solve from memory.
2. Day 7: solve under time.
3. Day 21: final validation.

---

## 4) Monthly Milestones

1. Month 1: 30 core problems.
2. Month 2: 60 core problems.
3. Month 3: 90 core problems.
4. Month 4-5: 120 core problems.
