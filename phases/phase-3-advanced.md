# Phase 3 - Advanced Mastery (Optional, 6-12 Weeks)

> Only do Phase 3 if you are targeting top-tier companies or have 6+ months.

---

## 1) Outcome You Must Reach

1. Comfort with advanced data structures (segment tree, BIT).
2. Ability to solve hard DP and graph problems with correct modeling.
3. Speed + accuracy on medium/hard interview questions.

---

## 2) Choose Your Scope

Pick only what your target companies ask:

- Product companies: advanced DP + graphs.
- Competitive programming: full spectrum.
- Startups: fewer advanced topics, more system design.

---

## 3) Core Topics (Pick 4-6)

1. Segment Tree + BIT
2. Advanced DP (digit, bitmask, tree DP)
3. String Algorithms (KMP, Z, Trie applications)
4. Advanced Graphs (SCC, bridges, articulation points)
5. Geometry or math (only if required)

---

## 4) Weekly Execution Loop

1. 2 concept days (read + 2 problems each).
2. 2 problem-heavy days (2 mediums or 1 hard).
3. 1 mixed day (random pattern).
4. 1 review day (journal + re-solve).

---

## 5) Phase 3 Gate

You are done when you can:

1. Solve one hard problem with correct modeling.
2. Explain advanced tradeoffs clearly.
3. Maintain 45-min interview communication quality.
