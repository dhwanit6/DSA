# Start Here - Your First 48 Hours

> If you feel lost, follow this page exactly. No guessing.

---

## 1) Who This Guide Is For

This guide is for students preparing for coding interviews and placements.

You should already know basic programming:

- variables, loops, and functions
- arrays/strings input-output
- one language: C++, Java, or Python

If this base is missing, spend 2-3 weeks on programming basics first.

---

## 2) The Interview Rubric You Must Train

Interview success is not only "correct code." You are scored on 4 dimensions:

1. Communication: explain your thinking while solving.
2. Problem Solving: start from brute force and optimize.
3. Technical Quality: complexity, clean code, edge cases.
4. Testing: dry run and bug catching before final answer.

<Flashcard question="What should I do in every practice session?">
Speak your approach aloud, state complexity, and test with one normal + one edge case.
</Flashcard>

## 5) Non-Negotiable Study Rules

1. 30-minute struggle before hints.
2. One short journal entry after each solved problem.
3. One timed mock every week after Phase 1 midpoint.
4. Daily minimum 90-minute focused block.
5. CS fundamentals are for structured weekly slots, not panic-night cramming.

## 6) If You Feel Overwhelmed

1. Do only today's planner tasks.
2. Ignore future chapters for now.
3. Return tomorrow and repeat.
