# Sorting Algorithms - What You Actually Need

---

## 1) What Matters for Interviews

1. Know how `sort` works conceptually (introsort).
2. Be able to write merge sort and quicksort once.
3. Know stable vs unstable sorting.

---

## 2) The Short List

- Merge sort (stable, O(N log N))
- Quick sort (average O(N log N), worst O(N^2))
- Heap sort (O(N log N))
- Counting sort (when range is small)

---

## 3) Practice Tasks

1. Implement merge sort from memory.
2. Implement partition logic for quicksort.
3. Solve 2 problems using sorting as main step.
