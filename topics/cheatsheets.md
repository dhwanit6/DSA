# Cheatsheets - Fast Review Pack

---

## 1) How to Use

1. Read once before mocks.
2. Use for pattern recall, not learning.
3. Add your own notes after each mock.

---

## 2) Must-Have Sheets

- Sliding window
- Two pointers
- BFS/DFS
- DP templates
- Binary search variants

---

## 3) Weekly Task

1. Pick 2 patterns and write your own summary.
2. Use them in a timed mock.
