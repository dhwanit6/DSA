# Complexity Analysis - The Skill That Separates Medium from Hard

---

## 1) Core Idea

You must estimate time and space before you code. It is part of the interview score.

---

## 2) Daily Use Checklist

1. Identify loops and nesting.
2. Identify recursion depth.
3. Find data structure costs.
4. State time + space out loud.

---

## 3) Quick Rules

- Two nested loops over N -> O(N^2)
- Sorting -> O(N log N)
- Hash map average -> O(1)
- Recurrence -> use Master Theorem or expansion

---

## 4) Practice Tasks

1. Analyze 5 solved problems and write time/space.
2. Pick one recursion problem and derive recurrence.
3. Build your personal complexity cheat sheet.
