# Pattern Recognition - The Trigger Map

---

## 1) Core Idea

Recognize the pattern before you code. This saves time and prevents wrong starts.

---

## 2) Trigger Table

| Trigger | Pattern |
|---|---|
| "Longest/shortest subarray" | Sliding window |
| "Pair/triplet in sorted array" | Two pointers |
| "First valid value" | Binary search on answer |
| "Count ways" | DP |
| "Process levels" | BFS |
| "Next greater/smaller" | Monotonic stack |

---

## 3) Practice Drill

1. Take 10 random problems.
2. Before reading fully, guess the pattern.
3. After solving, validate or correct your guess.
