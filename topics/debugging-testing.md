# Debugging and Testing - The Hidden Score

---

## 1) The 5-Step Debug Loop

1. Reproduce with smallest input.
2. Print key variables.
3. Validate assumptions.
4. Fix and re-run.
5. Add edge case test.

---

## 2) Edge Case Checklist

- Empty input
- Single element
- All same values
- Max constraints
- Negative values (if allowed)

---

## 3) Practice Tasks

1. Take 3 solved problems and add edge-case tests.
2. Intentionally insert a bug and fix it.
3. Practice using debugger once (VSCode or GDB).
