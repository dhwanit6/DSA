# OA Strategy - The First Gate

> Online assessments are where most candidates get filtered. Treat them as a separate skill.

---

## 1) OA Rules That Matter

1. Scan all problems first.
2. Solve the easiest fully before starting medium.
3. Track time: 30/30/40 split for 3 problems.
4. Always pass sample tests before moving on.

---

## 2) Constraints -> Complexity Map

| Constraint | Likely Complexity |
|---|---|
| N &lt;= 10^5 | O(N log N) or O(N) |
| N &lt;= 10^3 | O(N^2) possible |
| N &lt;= 10^2 | O(N^3) possible |

---

## 3) OA Execution Plan

1. 5-minute scan and ordering.
2. Solve easy first (score lock-in).
3. Solve medium with clear explanation.
4. If time remains, attempt hard with partial points.

---

## 4) Practice Protocol

1. One timed OA every week.
2. Webcam on. No breaks.
3. Write a 3-point debrief: mistake, fix, next time.
