## The Company Matrix

| Company | DSA Level | What They Really Test | Special Preparation |
|---|---|---|---|
| **Google** | Hard (Hards regularly) | Clarity of reasoning. Clean code. Optimality. | Practice TALKING through your solution. Google evaluates communication as heavily as correctness. Write clean, well-named code. |
| **Meta** | Medium-Hard (Speed) | Speed + correctness. AI-copilot round (pilot). | Practice mediums in < 20 min. Prepare for AI-assisted coding format. See [AI-Era Mastery](../topics/ai-era-mastery.md). |
| **Amazon** | Medium | Leadership Principles weighted EQUALLY with technical. | Prepare 16 LP stories in STAR+ format. For tech: trees, graphs, DP, OOD (parking lot, LRU cache). See [LLD/OOD](../topics/lld-ood-design.md). |
| **Microsoft** | Medium | Collaborative problem solving. OOD often appears. | Be very collaborative. Practice OOD: design patterns, class hierarchies, SOLID principles. |
| **Atlassian** | Medium | Real-world engineering judgment. Code quality. | Focus on code quality above all. Variable naming, function decomposition, edge cases — explicitly evaluated. |
| **Indian Product** (Flipkart, Swiggy, Meesho, PhonePe) | Medium + System Design | System design expected at fresher level. SQL sometimes. | Prepare system design basics. Know SQL joins and aggregations. |
| **Mass Recruiters** (TCS, Infosys, Wipro, HCL) | Easy-Medium + Aptitude | Verbal aptitude + quantitative + basic DSA + SQL. | Practice quantitative aptitude (GeeksForGeeks, IndiaBix). |
| **Research Labs** (Google Research, DeepMind, OpenAI) | Hard + ML depth | ML fundamentals + coding + research thinking. | Know ML basics: gradient descent, backpropagation, bias/variance. Publications help enormously. |

## Reading Recent Interview Experiences — The Most Underused Strategy

Interview questions repeat. Patterns and topics repeat heavily within a company within a 6-12 month window.

**Where to read (in priority order):**
1. **LeetCode Discuss** — filter by company, sort by recent. Read the last 6 months.
2. **GeeksForGeeks Interview Experiences** — extensive archive, especially Indian companies.
3. **Glassdoor Interview Questions** — filter by company, role, "software engineer."
4. **r/developersIndia, r/cscareerquestionsIN** — real recent experiences from Indian students.

> ✅ Spend the final 2 weeks before any specific company's drive doing ONLY that company's tagged problems and reading their recent interview experiences.
