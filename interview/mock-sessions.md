# Mock Sessions - How to Practice Like Interview Day

> Mocks convert knowledge into execution. Start early.

---

## 1) Minimum Mock Schedule

- Phase 1 mid-point: 1 mock every 2 weeks.
- Phase 2 and beyond: 1 mock per week.
- Last 2 weeks before interview: 2-3 mocks/week.

---

## 2) 45-Min Mock Script

1. 0-3 min: clarify.
2. 3-8 min: brute force + optimize path.
3. 8-30 min: code while narrating.
4. 30-35 min: test with edge cases.
5. 35-45 min: summarize + next steps.

---

## 3) Mock Debrief Template

1. What did I miss?
2. What was slow?
3. What will I change next time?
