## How to Use This List

- Do problems **in topic order**, not randomly
- Spend **30-45 minutes minimum** on each before looking at hints
- After solving, write a **[Patterns Journal](../topics/patterns-journal-template.md) entry**
- Problems marked `[Both]` = priority for both internship and full-time preparation
- Problems marked `[FT]` = priority for full-time; still useful for internship but lower priority
